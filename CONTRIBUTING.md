** How to Contribute **
- Step 1: Create a fork of dbadventtruth/dbat's develop branch
- Step 2: Clone repo to your local developent environment. VS Code is recommended.
- Step 3: Create a branch based on latest develop.
- Step 4: Edit and test your code. Add your changes to `CHANGELOG.md`. Commit changes to your branch.
- Step 5: Push your branch to your forked repo.
- Step 6: Create a Pull Request from your fork:branch to dbadventtruth/dbat:develop

** Note on Rebasing **
Because the develop branch will be updated periodically as Pull Requests are approved and merged, it's wise to continuously update and rebase your fork and local develop branches.
